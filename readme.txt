Extensional Functionalities for WP Theme ShapeShifter.

== Description ==

Extensions for WP Theme ShapeShifter.

<ul>
	<li>Custom Widgets</li>
	<li>Optional Widget Areas</li>
	<li>HTML Insert into Head, Header, Footer.</li>
	<li>Custom CSS( Uploading Files Available )</li>
	<li>Custom Font( Uploading Files Available )</li>
	<li>SS Button on Editor</li>
	<li>Theme Settings Page</li>
</ul>


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wp-theme-shapeshifter-extensions` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress


== Frequently Asked Questions ==

= how to set up? =

Go to "Appearances" -> "Appended Admin Pages".
